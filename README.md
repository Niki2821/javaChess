# javaChess

A terminal-based chess game written in Java. This personal project lets you play chess directly in your terminal window. The game logic is implemented in Java, and no graphical interface is required.

## Features

- Play standard chess against another player in the terminal
- Simple and user-friendly interface
- Every move is validated
- Special moves like castlig are supported
- This does use linux terminal colors to show teams

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Niki2821/javaChess.git
   ```
2. Compile the code:
   ```bash
   javac *.java
   ```
3. Run the game:
   ```bash
   java Main
   ```
